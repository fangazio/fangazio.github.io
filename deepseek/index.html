Deepseek简介

Deepseek是一种人工智能技术，用于信息检索和文本分析。它可以帮助用户快速找到他们感兴趣的内容，并提供相关的搜索建议和提示词。Deepseek的使用方法简单直观，适用于各种领域和应用场景。无论是学习、工作还是娱乐，Deepseek都可以提供准确、高效的搜索结果。

而对于消息来源是朋友圈，微博或者其他非专业领域媒体的大家来说，只要知道它是一个非常流弊的国内自主研发的人工智能大模型就好了！它的热度可比当晚稀碎的春晚高多了！

如何使用

  • 直接官方使用 ai.com (💡ai.com这个”天价”域名都买了,之前接入的是chatgpt！)

  • 其他非官方已接入Deepseek的网站（主流非主流的貌似都已经接入了）。

本地部署

部署到本地，就可以随时随地的轻松使用Deepseek,而且可以使用无限制版本的模型，我们国内使用的模型都是审核过的，所以加了限制。

使用本地模型，要注意自己的电脑配置，满血的671b模型，你想顺畅运行的成本大概是要50万美金！这只是你运行的基础核算，还不包括后期的维护等等成本，当然作为个人使用的话，也不需要投入这么大成本，直接在自己配置内使用可以流畅运行的模型或者直接网页端使用就好！

本地部署的话，要用到ollama这个东西，去ollama.com下载自己相应系统的版本，然后直接终端里下载你要用的模型即可，这里的ollama默认保存路径是C盘（MacOS用户无需担心这个），如果存储不是很富裕的话，可以把默认下载模型路径设置到其他盘，设置如下：

你如果不知道自己的设备可以运行多大的模型，可以用这个网站来检测一下，看看自己的设备能够运行多大的模型:

ollama的官网有很多模型，根本自己的需求去下载，下载也很简单，打开终端输入你要下载的模型的命令，比如deepseek-r1就是ollama run deepseek-r1,等待下载完成就可以使用了，本身在终端就可以使用，但是UI界面不友好。

可以使用chatbox，cherry studio，anything llm，LM studio等UI体验友好的软件，软件很多，根据自己喜好去使用。MacOS上本地部署好了后，搭配Bob使用相当丝滑！！！

很多高阶用法，有待自己去探索，如只是简单的了解使用，到这里基本差不多了。也不用看小红书抖音那些让你焦虑的标题党视频，徒增烦扰，他们目的是卖课，卖的也是全网已知的那些！
